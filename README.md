# Twister
Control and data visualization tool for twist machine "Twister" at Metamaterials LAB.

## Embed Install:

Using MSYS2 environment:
- arm-none-eabi-gcc
- openocd
- make

## Install: 
- https://mark-10.com/resources/software-drivers/
- PyQT5 module
- Serial module
